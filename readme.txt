== Description ==
Display automatically updating latest tech news widget in your sidebar your content will change so this will help you with SEO as googlr will see your site updates 

== Installation ==

* Download
* Extract and copy Tech News folder to Plugins Directory of your blog. (do not rename!)
* Activate Plugin
* Go to widget and drag/add Tech News widget to your Sidebar
* Done!
